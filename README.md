# 🛠️ Evyatar Hazan - Data Conversion Tools

Welcome to my collection of free, fast, and privacy-focused data conversion tools. Each tool is designed to work entirely in your browser, ensuring your data never leaves your computer.

---

## 🚀 Available Tools

### 📄 JSON to CSV Converter
Transform your JSON data into clean, structured CSV files.
- **URL**: https://json-to-csv.evyatarhazan.com/
- **Features**: Instant conversion, handles quoted fields, privacy-first design
- **Use Case**: Export API responses, data analysis, spreadsheet imports

### 📊 CSV to JSON Converter
Parse your CSV and convert it into structured JSON.
- **URL**: https://csv-to-json.evyatarhazan.com/
- **Features**: Header-based parsing, handles missing fields, privacy-first design
- **Use Case**: API workflows, data transformation, modern app integration

---

## 🛠 Tech Stack

All tools are built with **vanilla technologies** for speed and simplicity:
- **HTML5**: Semantic structure for accessibility and SEO
- **CSS3**: Modern styling with variables, Flexbox, and responsive design
- **JavaScript (ES6+)**: Pure JS for logic, no frameworks or dependencies
- **GA4**: Optional analytics integration

## 🚦 Getting Started

### View Locally
Open `index.html` in your browser or use a local server:

```bash
# Using Node.js serve
npx serve .

# Or using Python
python3 -m http.server 8000
```

Then visit http://localhost:8000
